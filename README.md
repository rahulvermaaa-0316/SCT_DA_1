# SCT_DA_1 - Superstore Sales Dashboard

## Objective
Analyze the Superstore Sales dataset using Microsoft Excel and create an interactive dashboard with Pivot Tables, Charts, and Conditional Formatting.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Dashboard Components
1. Profit by Category
2. Sales by Region
3. Sales Trend by Year
4. Key Insights

## Key Insights
- Technology category generated the highest profit (145,454.9).
- Furniture category generated the lowest profit (18,451.3).
- West region contributed the highest sales (725,457.8).
- Sales increased steadily from 2015 to 2017, indicating business growth.
